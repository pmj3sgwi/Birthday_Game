# Game-BTTF — 復古 2D 冒險遊戲（Back to the Future 主題）

一款以《回到未來》（Back to the Future）為靈感的 Pygame 復古 2D 冒險遊戲，玩家可以在不同場景中探索、收集道具、操作時間旅行日曆，並體驗迷你格鬥遊戲。

---

## 🎮 目前已完成的功能

### 1. 復古像素風格渲染
- 使用低解析度虛擬畫布（320×240）放大到視窗解析度（960×720），產生經典的像素顆粒感
- 1988 年場景使用更低的解析度（160×120），營造更加復古的視覺氛圍

### 2. 雙場景系統
- **客廳（Living Room）**：包含桌子、電子日曆、電視、櫃子、通往房間的門
- **房間（Bedroom）**：包含書櫃、電腦桌（可進入迷你遊戲）、通往客廳的門
- 玩家可透過門在兩個場景間自由切換

### 3. 互動式電子日曆
- 桌上的日曆可用方向鍵分別調整年、月、日
- 設定特定日期（如 2026/6/22）會解鎖場景內的互動物件
- 設定為 **1988/6/22** 會觸發時空旅行事件，進入 1988 年的客廳

### 4. 1988 年時空旅行場景
- 進入後場景全黑，玩家需使用手電筒道具照亮周圍環境
- 手電筒會產生圓形光圈效果，只能看到光圈範圍內的物件
- 目標是找到並啟動電燈開關，點亮整個房間
- 開燈後可看到完整的 1988 年客廳（泛黃舊色調）

### 5. 電視互動
- 靠近電視按空白鍵可觀看電視
- 顯示《回到未來》(Back to the Future) 電影圖片

### 6. 櫃子 / 抽屜系統
- 櫃子有上下兩個抽屜
- **上層抽屜**：可開啟並取得「手電筒」道具
- **下層抽屜**：上鎖狀態，需要「鑰匙」道具才能解鎖

### 7. 道具欄系統（Inventory）
- 畫面底部有 5 格道具欄
- 使用數字鍵 1~5 切換選中的道具格
- 目前可收集的道具：
  - 🔦 **手電筒（Flashlight）**— 從櫃子上層抽屜取得，用於 1988 年黑暗場景
  - 🔑 **鑰匙（Key）**— 通過 Street Fighter 迷你遊戲獲得，用於解鎖櫃子下層

### 8. Street Fighter 迷你遊戲
- 在房間的電腦上可啟動即時格鬥遊戲
- **玩家操作**：方向鍵移動、↑ 跳躍、空白鍵攻擊
- **CPU 對手**：Zangief AI，會自動追蹤玩家並發動攻擊
- 採用 **Best of 3** 制（先贏 2 回合獲勝）
- 贏得比賽後可獲得「鑰匙」道具

### 9. 碰撞偵測與互動提示
- 玩家靠近可互動物件時，會在頭上顯示操作提示文字
- 物件碰撞偵測防止玩家穿過家具

### 10. PyInstaller 打包支援
- 包含 `.spec` 設定檔，可將遊戲打包為獨立的 Windows 執行檔
- 資源路徑使用 `sys._MEIPASS` 相容 PyInstaller 打包環境

## 🚀 如何執行

### 環境需求
- Python 3.x
- Pygame

### 安裝與啟動
```bash
pip install pygame
python retro_game.py
```

### 打包為執行檔
```bash
pip install pyinstaller
pyinstaller retro_game.spec
```

## 🎯 操作方式

| 按鍵 | 功能 |
|------|------|
| `WASD` / 方向鍵 | 移動玩家 |
| `空白鍵` | 與物件互動 |
| `ESC` | 關閉介面 / 返回 |
| `1~5` | 切換道具欄格子 |
| `↑` (迷你遊戲中) | 跳躍 |
| `空白鍵` (迷你遊戲中) | 攻擊 |
